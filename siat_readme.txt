IP地址(内网IP):
    一共十台服务器,IP地址为 192.168.200.41 ~ 192.168.200.50

登陆方法:
    假设 fst_rsa 和 fst_rsa.pub 存放在 ~/.ssh/ 目录下面, 登陆命令(在本地打开终端)为:

    ssh -i ~/.ssh/fst_rsa fst@IPADDR

        @param fst  登陆用户名,已经提升为 root 权限,不需要密码.
        @param IPADDR IP地址,请从上面的列表 [IP地址] 中选取.
注释:
    为了方便,通过任意一台服务器,都可以通过 ssh 远程登陆其他服务器, 登陆方法同上.
    
    config是配置文件的样例, 是为了更方便地登录服务器, 可根据具体情况修改
    例如配置完成之后登录 192.168.200.41 只需在终端输入 "ssh 41" 即可
